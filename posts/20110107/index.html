VS2008安装出现严重错误1063错误

发表于2011-01-07|更新于2026-05-31|web开发
|浏览量:

今天在windows server 2003上安装VS2008时出现错误，VS2008上显示的是

出现严重错误，出现**.net framework 3.5 安装错误**出现错误，查看日志文件显示的

是1063错误，上网查资料，解决方法有以下几种：

1、IIS清理工具，清理IIS后重装。（此方法没试过）

2、下载单独的.net framework 3.5 进行安装，然后安装VS2008

3、查看自己在安装VS之前是否启动过一些服务（如sql Server 数据库，其他版本的VS（2005））

如果有的话，将其关闭，因为有些服务的开启为阻碍.net framework 3.5 的安装（我就是这种情况）

通过上述方法应该就能解决问题了。
