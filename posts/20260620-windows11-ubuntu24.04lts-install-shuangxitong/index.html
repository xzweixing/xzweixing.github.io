成功安装Windows11+Ubuntu24.04 LTS 双系统

发表于2026-06-20|更新于2026-06-21|生活随笔
|浏览量:

近期帮助朋友开发一个平台，windows系统在跑项目时总是有各种环境和Linux服务器不兼容的问题，于是准备安装Windows11+Ubuntu24.04 LTS 双系统。

一、前期准备工具

1、U盘：≥8GB 空白U盘，优先USB3.0接口，读写速度更快，装机耗时更短

2、网络：稳定有线/WiFi，用于下载系统镜像、后期装机驱动

二、工具+镜像下载地址（官方无毒）

1、Ubuntu24.04镜像下载 官网地址：Ubuntu官方桌面版下载 选择 Ubuntu 24.04 LTS Desktop，后缀.iso镜像，不要下载服务器版

2、启动盘制作工具 Rufus 官网地址：Rufus官方下载 推荐版本：4.13 稳定版，无广告、无捆绑、适配UEFI双系统

三、Rufus制作Ubuntu24.04启动U盘

1、 插入备好U盘，右键管理员身份打开Rufus，避免权限写入失败

2、设备选项：下拉选中自己的U盘（核对容量，选错会清空硬盘）

3、引导选择：点击【选择】，选中下载好的Ubuntu24.04 ISO镜像文件

4、 分区方案：固定选择 GPT；目标系统类型：UEFI(非CSM)，适配Win11原厂分区

5.、文件系统：默认FAT32，簇大小默认即可，无需修改

6、底部高级选项：取消【添加修复文件】，勾选【使用ISO镜像模式写入】，最大限度保留系统原生引导

7、点击右下角【开始】，弹窗全部确认点击【确定】，等待5-8分钟写入完成，提示就绪后安全弹出U盘

四、Windows11磁盘管理压缩分区

原理：必须提前在Windows划分空白空间，Ubuntu占用，不可安装在Windows已有分区内。

1、 Win11桌面右键左下角开始菜单 → 点击【磁盘管理】

2、找到系统硬盘，识别特征：带有 EFI系统分区、GPT分区表

3、选中剩余大容量空闲分区，右键【压缩卷】

4、压缩空间填写：157200 MB（150GB），点击压缩

5、压缩完成后，出现黑色未分配空间，禁止右键新建简单卷，保持空白即可！

红线避坑：千万不要格式化、分配盘符，一旦新建分区，Ubuntu无法识别空闲空间，会破坏Windows引导！

五、Ubuntu24.04LTS 桌面安装

本小节为装机核心，重点标注分区、引导位置两大红线，照着点击，不会弄坏Windows系统。

第一步：U盘启动进入安装界面

1、插入做好的启动U盘，重启电脑

2、开机连续按 F11 调出微星快捷启动菜单

3、选择选项：UEFI: 你的U盘品牌，禁止选择Legacy开头U盘

4、进入Ubuntu紫色安装主页

第二步：基础界面配置（无脑下一步）

1、语言：左侧下滑选择【简体中文】→ 点击【安装Ubuntu】

2、键盘布局：默认汉语(中国) → 下一步

3、网络设置：连接有线/WiFi，两个勾选全部打勾 ✅ 安装时下载更新 ✅ 安装第三方图形、声卡、硬件驱动

4、安装类型【重中之重】：选择 其他选项(手动分区) ❌ 禁止选择：安装Ubuntu alongside Windows（自动分区破坏Win引导）

第三步：150GB未分配空间手动分区（照搬大小即可）

选中页面底部黑色【未分配】空间，依次点击【+】新建分区，顺序不能乱：

优点：单独划分/home分区，以后重装 Ubuntu 不会丢失项目代码、大模型文件

1、安装类型页面选择「手动分区」，进入磁盘界面
2、找到磁盘 0 下150GB 未分配空间，点左下角「+」新建 3 个分区：
1）交换分区 swap
大小：16G 内存填16384MB；32G 内存填8192MB，用途选择「交换空间」
2） 根分区 /
大小：60000MB，文件系统ext4，挂载点选 /
存放系统、PHP/Node/MySQL 环境、Docker 镜像、显卡驱动
3）/home 数据分区
剩余全部空间（约 80GB），文件系统ext4，挂载点 /home
存放项目源码、本地大模型、数据库备份，重装系统仅格式化根分区，此分区数据保留
3、页面底部「用于安装引导程序的设备」下拉选择磁盘 0（整块硬盘，共用 Windows 自带 EFI 分区）
4、 确认分区后点击下一步，继续时区、用户名配置

⚠️ 关键禁止操作
  • 不要右键这块 150GB 未分配空间「新建简单卷」，会变回 Windows NTFS 分区，Ubuntu 无法识别；
  • 全程禁止删除 / 格式化 C 盘、D 盘，否则 Windows 系统损坏；
  • 引导设备务必选整块磁盘磁盘0，不要单独选分区，避免开机无双系统启动菜单。

第四步：引导安装位置（决定能否双系统开机）

页面最下方：用于安装引导程序的设备

下拉选择：带有 windows boot manager 的EFI分区，绝对不要选U盘

第五步：收尾安装设置

1、 点击【现在安装】，弹窗确认写入磁盘，点击继续

2、地区时区：选择上海，下一步

3、账号设置：用户名、自定义主机名，设置开机密码

4、等待系统拷贝文件10分钟，安装完成拔掉U盘，点击重启

✅ 装机完成效果

重启自动进入Grub开机选择界面，可自由切换：Windows11 / Ubuntu24.04，双系统安装完毕。
