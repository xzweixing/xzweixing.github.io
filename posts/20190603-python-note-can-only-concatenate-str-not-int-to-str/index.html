[Python笔记]can only concatenate str (not &quot;int&quot;) to str

发表于2019-06-03|更新于2026-05-31|Python
|浏览量:

代码：

a=input(“请输入乘数：”)
b=input(“请输入被乘数：”)
print(“您输入的是”+a+”乘以”+b+”等于”+int(a)*int(b))

错误提示：

can only concatenate str (not “int”) to str

错误原因：输出的类型必须一致，前面是str字符串，后边是int整数，所以错误，需要转换类型

正确代码：

a=input(“请输入乘数：”)
b=input(“请输入被乘数：”)
print(“您输入的是”+a+”乘以”+b+”等于”+str(int(a)*int(b)))
