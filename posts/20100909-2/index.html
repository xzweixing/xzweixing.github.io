截取不定长度的字符串！！！

截取“src=””到“””的字符串！（src=” ********” ?? 星号为所要截取的）

string getsrc(string banner)

{

int starti = banner.IndexOf(“src=””);

banner = banner.Substring(starti + 5);

int endi = banner.IndexOf(“””);

banner = banner.Substring(0, endi);

return banner;

}

其中 “ 只代表 “? ,加上 ? 是过滤掉 “ “ 冲突！！
