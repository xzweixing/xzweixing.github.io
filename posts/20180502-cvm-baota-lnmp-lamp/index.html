腾讯云CVM安装宝塔面板及LNMP/LAMP环境配置

发表于2018-05-02|更新于2026-05-31|科技频道
|浏览量:

前几天新购腾讯的CVM云服务器一台，安装了宝塔的管理面板以及LNMP/LAMP环境，现在将安装方法分享给大家，其实阿里云的ECS的安装方式也是一样的，下面是详细步骤：

1、使用远程登录软件连接我们的服务器。

采用密码登录 Linux 实例（本例中选择使用** PUTTY**，大家也可以选择其他类型的登录软件，推荐下载地址：https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html）。

登录默认账户为root，密码为你之前设置的密码（此时输入密码是不显示任何内容的，但你确实已经输入）

2、使用命令行安装宝塔面板

[su_highlight]yum install -y wget && wget -O install.sh http://download.bt.cn/install/install.sh && sh install.sh[/su_highlight]

你可以直接复制上面代码，然后鼠标右击就直接粘贴上去了，点击y开始安装

安装完毕后给出登录账户密码

3、登录BT面板。通过您的IP地址+8888端口号打开登录界面。进入后提示您安装LNMP/LAMP环境

最后为了安全起见，我们设置一下登录端口和账号密码，绑定自己的域名方便登录
