wordpress小工具中的标签文字大小修改

发表于2012-09-12|更新于2026-05-31|wordpress
|浏览量:

今天修改一个wordpress主题，需要让标签的字体大小确定，与页面其他链接相同，而不是有大有小，但是我是使用的小工具自带的标签，所以不能用css直接修改，于是在网上找了些资料，解决方法很简单，只要修改include下的一个文件的一小段代码就Ok了，下面是解决方法！！

wordpress文件中找到/wp-includes/category-template.php，搜索：wp_tag_cloud，则看到如下默认设置：

‘smallest’ => 8, ‘largest’ => 22, ‘unit’ => ‘pt’, ‘number’ => 45’，

‘format’ => ‘flat’, ‘separator’ => “n”, ‘orderby’ => ‘name’, ‘order’ => ‘ASC’,

说明：

  1. smallest – 定义标签的最小字号

  2. largest – 定义标签的最大字号

  3. unit – 设置字号类型，如 “pt” 或 “px”

  4. number – 设置标签云数量

  5. orderby – 设置按 “name” 或 “count” 排序，默认为 “name” 方式；

（注：orderby=count 表示按照标签使用次数排列）

  1. order – 设置按 “DESC” 或 “ASC” 升降序排列

我自己是将最大字号与最小字号都改成10，具体看你自己的需求了，OK，问题解决！！
