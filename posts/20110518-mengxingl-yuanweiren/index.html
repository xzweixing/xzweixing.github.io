除非 Windows Activation Service (WAS)和万维网发布服务(W3SVC)均处于运行状态，否则无法启动网站
好久没在自己的机子上做.net程序了，今天想做点东西，我是win7系统，配置IIS时出现错误。 错误提示：“除非 Windows Activation Service (WAS)和万维网发布服务(W3SVC)均处于运行状态，否则无法启动网站”。 解决方法：在WINDOWS服务管理里 启动?World Wide Web Publishing Service 然后出现以下错误： ? ?“HTTP 错误 403.14 – Forbidden Web 服务器被配置为不列出此目录的内容。” 提示原因：“没有为请求的 URL 配置默认文档，并且没有在服务器上启用目录浏览。” 解决方法：1）双击“目录浏览”，点击右边的“启动”按钮 2）双击“默认文档”，点击右边的“启动”按钮
