学做wordpress主题过程中，想新建一个页面，用来显示某个分类下的文章，文章不多，用来展示用的， 文章的标题做列表，点击实现选中效果，同时查看这篇文章的内容（展示的内容）。 我对wordpress中的结构不是很清楚，所以只有使用笨方法，思路是使用javascript获得网页URL地址，根据文章的URL来判断现在显示的是那篇文章，然后使用javascript给相应的文章的标题的li标签样式，获得想要的效果。
