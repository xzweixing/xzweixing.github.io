javascript获得select的value以及text值

发表于2011-09-22|更新于2026-05-31|web开发
|浏览量:

使用javascript获得select选中后的value以及text的值

方法如下：

`

请选择客户客户11客户22

function copyValue()
{
var selectId = document.getElementById(‘select’);
document.getElementById(‘titleValue’).value=selectId.value;
document.getElementById(‘titleText’).value=selectId.options[selectId.selectedIndex].text;
}

`

在使用此段javascript代码的时候有个小插曲，当option 的value不存在时，Firefox和Chrome浏览器的value值显示就是text的值，而IE不显示任何值！此处的说明是为了让新手不要把value和text弄混了，不然会出现意想不到的错误，但是对于高手来说可以通过这个hack实现意想不到的效果。
